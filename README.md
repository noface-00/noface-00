## 🧱 The MORA Ecosystem

| Project | What it does | Stack |
|---|---|---|
| **CarMora** | Monorepo workshop management system for an industrial mechanical workshop | NestJS · Prisma · PostgreSQL · Expo / React Native |
| **CarMora Seals** | Hydraulic seals & John Deere parts distributor app | Expo / React Native · Supabase |
| **AppSy** | Multi-tenant systems manager — centralizes Google OAuth and routes shared microservices across the MORA apps | NestJS · OAuth2 · Microservices |
| **ServFacGmail** | Gmail invoice-scanning microservice — parses SRI Ecuador e-invoices, with Gemini API as fallback | NestJS · Gmail API · Gemini API |

> 🔒 Most of these are private business repos — happy to walk through the architecture if you ask.
